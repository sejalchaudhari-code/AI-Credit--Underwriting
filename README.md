# AI Credit Underwriting & Risk Assessment Platform

This project is an AI-powered credit underwriting dashboard built for a hackathon prototype.

The platform helps financial analysts evaluate company credit risk by analyzing financial data, extracting insights from uploaded documents, and generating an automated credit assessment report.

## Features

- Entity onboarding
- Financial document upload
- PDF financial data extraction
- AI credit risk prediction
- Risk meter visualization
- AI news risk scanner
- SWOT analysis
- PDF credit report generation
- Portfolio analytics dashboard

## Technology Stack

Frontend: Streamlit  
Backend: Python  
Database: SQLite  
Visualization: Plotly, Matplotlib  
Document Processing: PDFPlumber  
Report Generation: ReportLab  

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

Open browser:

http://localhost:8501

## Project Purpose

This project demonstrates how AI and data analytics can improve enterprise credit underwriting and automate risk assessment workflows.
